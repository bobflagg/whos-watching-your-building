# Mission

**Who's Watching Your Building?** exists to make NYC housing accountability possible.

HPD complaints, code violations, property registrations, 311 requests, DOB violations, and deed transfer records are all out there — but scattered across separate datasets that most people will never know how to connect. This project pulls them together into a single knowledge graph and adds an AI-powered assistant that helps users navigate and interrogate it, so that tenants, journalists, advocates, researchers, and government employees can simply ask what they need to know and get answers that hold landlords and city agencies accountable.

A critical part of that accountability is ownership transparency. ACRIS — NYC's deed transfer and mortgage recording system — contains every ownership transaction in the city, including the LLC and corporate entity names on each filing. When combined with complaint and violation history, it becomes possible to trace patterns across shell companies and reveal the real actors behind problem properties.

---

The core belief is that open data's real value isn't in any single dataset — it's in the connections between them. Housing accountability is a relationship problem: who owns what, what happened there, who was responsible, what came next. This project is built to answer exactly those kinds of questions.

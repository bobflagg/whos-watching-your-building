# Mission

**Who's Watching Your Building?** exists to make NYC housing accountability possible.

HPD complaints, code violations, property registrations, 311 requests, and DOB violations are all out there — but scattered across separate datasets that most people will never know how to connect. This project pulls them together into a single knowledge graph and adds an AI-powered assistant that helps users navigate and interrogate it, so that tenants, journalists, advocates, researchers, and government employees can simply ask what they need to know and get answers that hold landlords and city agencies accountable.

---

The core belief is that open data's real value isn't in any single dataset — it's in the connections between them. Housing accountability is a relationship problem: who owns what, what happened there, who was responsible, what came next. This project is built to answer exactly those kinds of questions.
